# DFSA Estimators
 
 ## Requisitos
 - Python 3
 - Matplotlib

## Simulações disponíveis
- Lower Bound

## Como rodar
1. Clone o projeto
2. No shell, rode:
   ```bash
   make run
   ```
   
     ou
     
   ```bash
   python main.py
   ```
3. Gráficos gerados são salvos na pasta `graph_plots/`
